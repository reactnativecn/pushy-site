产品简介

Pushy - 专业的热更新解决方案

现已全面支持鸿蒙系统以及 React Native 新架构！

Pushy (npm 包名: react-native-update (opens in a new tab) ) 是面向 React Native 应用提供的专业热更新服务，由 React Native 中文网 (opens in a new tab)独家研发并持续维护。我们致力于为全球 React Native 开发者提供最快速、最稳定、最经济的热更新体验。

🚀 为什么选择 Pushy？

极致的更新体验

  • 🎯 超小更新包：基于 bsdiff/hdiff 算法的增量更新，通常只需几十至几百 KB（相比其他全量更新服务的几十 MB，节省流量 99%+）
  • ⚡️ 极速分发：国内用户使用阿里云高速 CDN，国外用户智能分流至 Cloudflare，在各地区都使用覆盖率最好的 CDN 服务商，更新成功率高达 99.9%+
  • 🛡️ 安全可靠：内置崩溃回滚机制，出现问题自动回退，确保应用稳定运行
  • 🌐 全平台覆盖：完美支持 iOS Android, 且业内独家第一方支持 鸿蒙（HarmonyOS）
  • 🔍 深耕 React Native 生态：
    • ✅ Expo - 无缝集成
    • ✅ 新架构 - 第一时间跟进支持
    • ✅ Hermes - 完整支持字节码格式
  • 📖 核心逻辑完全开源：客户端、命令行工具以及管理界面代码都完全开源 (opens in a new tab)，接受社区审计，代码透明可信，你可以随时迁移到自建服务器或其他服务
  • 🔒 数据安全：可以自行插入数据埋点，完全掌控自己的数据
  • 💬 专业人工支持：提供付费专人技术支持，有问必答，确保你的应用稳定运行
  • 🏆 多年稳定运营：从 2016 年开始运营至今，服务众多知名企业和应用

简单易用

// 三行代码即可集成
import { UpdateProvider, Pushy } from "react-native-update";
 
const pushyClient = new Pushy({ appKey });
 
<UpdateProvider client={pushyClient}>
  <App />
</UpdateProvider>

内置多种更新策略，开箱即用：

  • 🔕 静默更新 - 后台自动下载，用户无感知
  • 💬 提示更新 - 友好的系统弹窗提醒
  • ⚙️ 自定义策略 - 完全可控的更新流程
